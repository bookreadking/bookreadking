# DDDモデリング実装ガイド 読書会

# 読書会の進め方
- [ドメイン駆動設計 モデリング/実装ガイド](https://booth.pm/ja/items/1835632)を読んでいきます
- 前半は各自で黙読して後半に質疑応答の時間を取ります
- 読むペースは1分/ページとします（適切でなければ適宜調整します）
- 質疑応答の時間は自由に話してもらって良いですし、ちょっとした感想などあえて喋るのが躊躇する場合などはSli.doに書き込んでもらってもOKです
